Шахматы на Pygame - краткое описание проекта
Это простая графическая реализация классических шахмат, написанная на Python с использованием библиотеки Pygame. Проект создан для обучения основам игрового программирования и демонстрирует работу с событиями, отрисовкой и логикой ходов.

Возможности
  Полноценная шахматная доска 8×8 с чередованием цветов клеток.
  Все фигуры (король, ферзь, ладья, слон, конь, пешка) с правильным начальным расположением.
  Поддержка поочерёдных ходов белых и чёрных.
  Визуальная подсветка выбранной фигуры (красным) и доступных ходов (серыми кружками).
  Реализованы базовые правила перемещения для всех фигур (кроме рокировки, взятия на проходе и превращения пешки).
  Управление мышью: клик по фигуре — выбор, клик по подсвеченной клетке — ход.

Технологии
  Язык: Python 3
  Графика: Pygame
  Интерфейс: оконное приложение

Как запустить
  Установите Pygame: pip install pygame
  Скачайте файл chess_game.py с исходным кодом.
  Запустите его: python chess_game.py

Планы по развитию
  Проект является базовой версией. В будущем можно добавить:
  Проверку шаха и мата
  Рокировку и взятие на проходе
  Превращение пешки
  Искусственный интеллект
  Звуковые эффекты и меню

Проект отлично подойдёт для изучения структуры игр на Pygame и может служить основой для более сложных шахматных приложений.
